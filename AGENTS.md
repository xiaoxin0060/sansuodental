# AGENTS.md - 三索齿科网站

## 项目概览

三索齿科官方网站，单页展示型网站，提供诊所信息、服务项目、医生介绍和联系方式。

## 技术栈

- **类型**: 原生静态网站 (native-static)
- **语言**: HTML5 + CSS3 + JavaScript
- **构建**: 无构建工具，纯静态文件
- **服务**: Python HTTP Server

## 项目结构

```
/workspace/projects/
├── index.html          # 主页面（单页应用）
├── styles/
│   ── main.css        # 主样式表
└── .coze               # 项目配置
```

## 页面模块

1. **导航栏** - 固定顶部，包含 Logo、导航链接、联系电话
2. **Hero 区域** - 诊所名称、标语、预约电话、地址和营业时间
3. **服务项目** - 9 个服务卡片网格布局
4. **医生介绍** - 贺三锁主治医师简介
5. **联系我们** - 电话、地址、营业时间
6. **底部** - 版权信息和快速导航

## 设计规范

### 配色方案
- 主色：蓝灰色 `#5a7a8a`（招牌底色）
- 辅助色：绿色 `#2d8a5e`、红色 `#d4382c`（牙齿 Logo）
- 背景：白色 `#ffffff`、浅灰 `#f8f9fa`

### 响应式
- 移动端优先设计
- 断点：768px（平板）、480px（手机）
- 移动端菜单：汉堡菜单

## 开发命令

```bash
# 启动开发服务
python -m http.server 5000 --bind 0.0.0.0

# 访问地址
http://localhost:5000
```

## 修改指南

### 修改诊所信息
编辑 `index.html` 中的对应文本内容

### 修改样式
编辑 `styles/main.css`，CSS 变量定义在 `:root` 中

### 添加服务项目
在 `.services-grid` 中添加新的 `.service-card` 元素

### 修改联系方式
搜索 `13994027312` 替换为新号码

## 注意事项

- 所有电话号码使用 `tel:` 协议，支持点击拨打
- 图片资源使用 SVG 图标，无需外部图片
- 纯前端实现，无需后端服务
- 部署时确保 `.coze` 文件配置正确
